Functions are also called Procedures. Procedures historically are a way of organising code in mainstream languages like Java, JS, PHP etc.

###### Unsolved software engineering problems
- Almost all software is hard to change
- Almost all software is hard to integrate

The reason software is hard to change is because there are fundamental business and technical assumptions all scattered and passed through different procedures/functions that make up a software component. 

##### RPC vs HTTP

Http APIs are easier to learn and reason about largely due to the Uniform interface provided by RESFUL HTTP implementations. There are universally similar methods (GET/POST) methods that mean the same thing across all http implementations.

RPC on the other hand has a steeper learning curve. There are no universal procedures/methods as procedures can be named anything and cause different side effects.

Learning RPC is like learning a programming language standard library.
